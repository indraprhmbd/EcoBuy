# EcoBuy

EcoBuy is a minimalist marketplace for industrial waste trading. It facilitates the buying and selling of recyclable materials with AI validation and environmental impact tracking.

## Core Features

* AI Validation. Automated grading and identification of industrial waste.
* Marketplace. Streamlined grid layout for trading recyclable materials.
* Impact Tracking. Measurement of CO2 savings from recycling activities.
* Performance. Lightweight mobile first design for fast loading and low data usage.

## Tech Stack

* Frontend. Vue 3, Vite, Tailwind CSS.
* Backend. FastAPI, Python.
* Infrastructure. Docker, PostgreSQL, AWS S3.

## Setup

### Development

1. Clone the repository.
2. Create .env file based on .env.example.
3. Run docker-compose up --build.

### Access

* Frontend. http://localhost:5173
* API Documentation. http://localhost:8000/docs
